# Examples

`examples` contains runnable CAD JSON jobs used by the README, Skill references, and unit tests.

## Layout

- `basic/`: small upper-template and entry examples such as mounting plates and L brackets.
- `primitives/`: direct `primitive_part` examples for core feature modeling, face references, revolve, and cut revolve.
- `advanced/`: path-driven and multi-section features such as sweep, cut sweep, twist sweep, guide-curve sweep, loft, and cut loft.
- `sketch/`: production sketch workflows such as sketch fillet/chamfer, slots, polygons, splines, convert/offset entities, trim/delete cleanup, and open edge slots.

Keep generated CAD outputs out of this folder. Runtime artifacts belong under `outputs/jobs/`.
