# Drawing Rules

MVP drawing output is a lightweight result report, not a full manufacturing drawing.

For future two-dimensional drawings:

- Use third-angle or first-angle projection consistently per project setting.
- Include front, top, right, and isometric views when useful.
- Show overall dimensions before detailed feature dimensions.
- Do not invent tolerances, GD&T symbols, surface finish, or material standards.
- Ask the user before adding manufacturing-critical annotations that were not specified.
